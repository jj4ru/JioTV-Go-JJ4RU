# 📺 JioTV Go - Ultimate Windows Launcher
**Developed by: jj4ru**

The **JioTV Go JJ4RU Player** is a highly optimized, automated, and professional Windows launcher built on top of the original JioTV Go core. While the standard installation traditionally requires PowerShell commands, this modified version provides a seamless **One-Click Execution** experience.

This launcher automates server initialization, performs network checks, and opens the TV interface directly in your default browser with a premium UI feel.

---

## ✨ Key Features
* 🚀 **One-Click Automation:** No more manual PowerShell commands; just run the `.exe` and start watching.
* 🖼️ **Professional Splash Screen:** A branded 'JJ4RU' startup screen that handles all background processes silently.
* 🌐 **Network Broadcasting:** Automatically detects your Local IP, allowing you to watch TV on any device (Mobile, Smart TV, Tablet) connected to the same router.
* 📶 **Smart Connectivity Check:** Real-time monitoring of your internet status during startup.
* 🔄 **Integrated Auto-Update:** Fetches the latest version details directly from GitHub to ensure you never miss an update.
* 📺 **Fullscreen Browser Sync:** Automatically launches your browser in maximized mode once the server is ready.

---

## 🛠️ How to Use
1. **Download:** Get the latest `JioTV_JJ4RU_Player.exe` from the releases.
2. **Launch:** Double-click the file. If Windows SmartScreen appears, click **'More Info'** -> **'Run Anyway'**.
3. **Wait for Splash:** The JJ4RU splash screen will initialize the server and check for updates.
4. **Stream:** Your browser will open automatically. To watch on other devices, use the IP address displayed on the splash screen (e.g., `http://192.168.1.5:5001`).
